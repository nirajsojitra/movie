# SN-Full HD Movies

# We will create a Complete movie website using HTML, CSS, and JavaScript. This HTML Movie website is fully responsive. This website depends upon two HTML pages one is the Home page the second is the Movie Page. On the Home page, you see an HTML Movie slider and some Latest movies Boxes. On the Movie Page, you saw a Trailer button(Go to Trailer), an Online play movie button (Play Full Movie Online), and Movie Download Buttons (Use to Download Movie).
